VLC Media Player

VLC Media Player 3.0.10 Final adalah sebuah software media player terbaru dari VLC yang dapat anda gunakan untuk memutar video dengan berbagai macam format dan dukungan penuh fitur multimedia yang sudah tersedia lengkap di dalamnya. Dengan menggunakan teknologi individual system codec, membuat VLC Media Player Terbaru ini dapat memainkan hampir semua format video dan audio yang ada saat ini. Anda juga dapat melakukan rekaman langsung menggunakan program media player yang satu ini.


Fitur


•Men-convert File Video maupun Audio
•Streaming
•Fitur Mouse Gestures
•Rekam Langsung Webcam


Cara instal


•Pertama, download dulu VLC di link ini.
•Setelah itu seperti biasa, langkah awal untuk menginstal sebuah aplikasi adalah melakukan dobel klik pada file installer aplikasi tersebut
•Jika muncul Security Warning, klik saja tombol RUN
•Pada pilihan bahasa penginstalan, pilih saja English, kemudian klik tombol OK
•Nah, pada Welcome To The VLC Media Player Setup Wizard, Klik tombol NEXT untuk melanjutkan penginstalan ini.
•Selanjutnya Anda berada pada tampilan License Agreement. Silahkan baca jika mau. Kalau tidak, langsung saja klik tombol NEXT
•Pada bagian Choose Components, pilih saja komponen-komponen apa yang akan Anda instal bersama VLC media player. Tapi kalau tidak mau memilih, klik saja tombol NEXT
•Sekarang, Anda berada pada pemilihan lokasi instalasi. Anda boleh memilih lokasi tempat diinstalnya aplikasi VLC dengan mencarinya menggunakan tombol BROWSE jika mau. Namun saran saya, biarkan saja secara default, agar aplikasi VLC terinstal pada tempat yang ditentukan oleh sistem ke dalam Drive C
•Nah, inilah saat yang ditunggu-tunggu.. Anda sekarang berada pada tahap akhir penginstalan VLC media player ke laptop. Pada tahap ini, klik tombol FINISH untuk mengakhiri penginstalan ini.
